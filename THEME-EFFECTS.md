# 海洋蓝色主题特效说明

博客已经更换为海洋蓝色主题，并添加了多种动态特效。

## 🎨 主题颜色

### 主色调
- **主题色**: #1e88e5（明亮的海洋蓝）
- **浅色**: #42a5f5
- **深色**: #1565c0

### 渐变效果
- 导航栏：紫蓝渐变（#667eea → #764ba2）
- 首页背景：海洋蓝渐变
- 按钮和卡片：柔和的渐变效果

## ✨ 特效功能

### 1. 点击特效
每次点击页面时会随机出现：
- ❤️ 彩色爱心
- 💙 海洋主题图标（💙🌊✨💫🌟⭐💎🔷）

特点：
- 随机颜色
- 上升渐隐动画
- 不影响页面交互

### 2. 背景粒子特效
页面背景有动态粒子效果：
- 50 个蓝色粒子在背景中浮动
- 粒子之间会自动连线
- 粒子会缓慢移动
- 鼠标靠近时粒子会被吸引

特点：
- 不影响页面性能
- 暗黑模式下自动调整透明度
- 响应式设计，适配所有屏幕尺寸

### 3. 首页海浪动画
首页 Hero 区域有动态海浪效果：
- SVG 海浪图案
- 15 秒循环动画
- 平滑的波浪起伏

### 4. 悬停动画
所有交互元素都有悬停效果：
- 导航链接：上移效果
- 侧边栏链接：右移 + 背景变化
- 特性卡片：上浮 + 阴影增强
- 按钮：立体感增强

## 🎮 自定义配置

### 关闭特效

如果需要关闭某些特效，可以编辑 `docs/.vuepress/client.js`：

```javascript
// 关闭点击特效：注释掉 ClickEffect
// 关闭背景特效：注释掉 BackgroundEffect

rootComponents: [
  // ClickEffect,     // 注释掉这行
  // BackgroundEffect, // 注释掉这行
],
```

### 调整粒子数量

编辑 `docs/.vuepress/client/BackgroundEffect.vue`：

```javascript
const particleCount = 50  // 修改这个数字
// 建议范围：30-100
// 数字越大，特效越密集，但性能消耗越大
```

### 修改点击特效图标

编辑 `docs/.vuepress/client/ClickEffect.vue`：

```javascript
const words = ['💙', '🌊', '✨', '💫', '🌟', '⭐', '💎', '🔷']
// 可以添加或替换为你喜欢的 emoji
```

### 调整颜色

编辑 `docs/.vuepress/styles/palette.scss`：

```scss
$color-brand: #1e88e5;  // 改成你喜欢的颜色
```

## 🚀 性能优化

所有特效都经过优化：

## 💡 使用建议

1. **首次访问**可能需要短暂加载特效组件
2. **低性能设备**可以考虑关闭背景特效
3. **移动端**特效会自动优化
4. **暗黑模式**下特效透明度会降低

## 🔧 故障排除

### 特效不显示

1. 清除缓存：`npm run clean`
2. 重启开发服务器：`npm run dev`
3. 检查浏览器控制台是否有错误

### 性能问题

1. 减少粒子数量（改为 30）
2. 关闭背景特效
3. 检查其他运行的程序

### 样式冲突

1. 清除浏览器缓存
2. 检查 SCSS 语法
3. 确保没有自定义样式覆盖
